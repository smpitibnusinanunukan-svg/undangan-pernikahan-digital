📱 Kirim Massal via WhatsApp

Kirim link undangan personal ke banyak kontak sekaligus. Setiap link mengandung nama tamu masing-masing.

🔗 Link Undangan

URL halaman undangan yang akan dikirim. Nama tamu akan otomatis ditambahkan sebagai parameter ?to=NamaTamu.

💡 Contoh hasil link: https://undangan.com/?to=Budi+Santoso
Nama akan tampil di layar pembuka dan hero undangan.

💬 Template Pesan

Gunakan placeholder: {nama} = nama tamu, {link} = link undangan personal, {no} = nomor urut.

📋 Daftar Kontak

🚀 Siap Publish ke Vercel?

Semua pengaturan Anda saat ini tersimpan di browser Anda (lokal). Agar tamu undangan lain bisa melihat desain yang sudah Anda buat, Anda harus mengekspor konfigurasi ini.

  1. Klik tombol Download config.js di bawah.
  2. Simpan file yang ter-download tersebut ke folder js/ di folder proyek Anda (menimpa file lama jika ada).
  3. Upload semua folder undangan Anda ke GitHub lalu deploy ke Vercel/hosting.
